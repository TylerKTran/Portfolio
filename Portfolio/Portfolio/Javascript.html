  HTML, CSS, and Javascript all go hand-in-hand. However, I did not have the same experience with all three of them.
  HTML is the simplest, easiest language I have ever learned. It is also so basic that it is virtually impossible to do anything in terms of design with it. That is where CSS comes in. CSS at heart is another easy language. However, it can just as easily get out of hand. Bootstrap is a tool for working with CSS. It is super useful when it works, and super frustrating when it goes wrong. Nevertheless, I have a good grasp of both HTML and CSS.
  Going in I thought that the client-side Javascript was going to be akin to Java. How wrong I was. Unfortunately, Ryan Bennett’s lecture on the whole language did not help my better comprehension. What did help were the in-class activities using vue. After completing those tasks, I had a better idea on how Javascript works and how it should be used.
  Altogether, I have a decent understanding of HTML, CSS, and Javascript, but I do not enjoy working with HTML and/or CSS due to my lack of “aesthetic vision.”
